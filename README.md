# Student-Suchana

Student-Suchana is a cloud-native student management platform designed using a microservices architecture.

The project enables educational institutions to manage students, teachers, attendance, and marks through independent services.

---

## Features

### Authentication Service
- Register Users
- Login Users
- JWT Authentication
- Role Based Access Control

### Student Service
- Create Student
- View Students
- Student Profile
- Student Count
- Update Student
- Delete Student

### Teacher Service
- Create Teacher
- View Teachers
- Teacher Profile
- Teacher Count
- Update Teacher
- Delete Teacher

### Attendance Service
- Mark Attendance
- Attendance Percentage
- Average Attendance

### Marks Service
- Upload Marks
- Student Percentage
- Top Performer
